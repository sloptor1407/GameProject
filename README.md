# EmberVeil

Videojuego 2D de plataformas y metroidvania desarrollado en Unity 6 como Proyecto Integrado del Ciclo Formativo de Grado Superior en Desarrollo de Aplicaciones Multiplataforma (DAM) en el IES Saladillo.

## Descripción

EmberVeil es un juego de plataformas 2D con mecánicas de metroidvania. El jugador debe explorar tres niveles progresivos, enfrentarse a distintos tipos de enemigos y derrotar a un jefe final con dos fases de combate.

## Mecánicas

- Movimiento, salto, doble salto, dash y salto en pared
- Combate melee y a distancia
- Mejoras de armas ocultas en zonas secretas
- Sistema de checkpoints y perfiles de jugador
- Base de datos SQLite para persistencia de datos

## Tecnologías

- Unity 6
- C#
- SQLite (sqlite-net-pcl)
- TextMeshPro
- Unity Input System

## Controles

| Acción | Tecla |
|---|---|
| Mover | A / D |
| Saltar / Doble salto | Espacio |
| Dash | Mayúsculas izquierda |
| Ataque melee | J |
| Ataque a distancia | K |
| Pausa | Escape |
